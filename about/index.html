About

Saya bersekolah di SMK Telkom Malang, saat ini saya berada di kelas X RPL 7. Untuk tugas akhir semester ini saya membuat web tentang rental mobil yang bernama MOBILKU.

Web ini dapat digunakan untuk meminjam berbagai macam mobil dengan disertai dengan form login dan ada transaksi di dalamnya pada saat peminjaman mobil. Jika pengembalian mobil telat, maka akan dikenakan biaya seharga Rp 100.000 per harinya. Masa peminjaman mobil paling lama adalah 3 hari saja.
